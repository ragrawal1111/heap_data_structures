# Assignment 4: Heap Data Structures

## Contents

- `heapsort.py`: Heapsort implementation
- `priority_queue.py`: Max-heap priority queue with task model
- `benchmark.py`: Empirical comparison against Quicksort and Mergesort
- `report.md`: Analysis and discussion

## Requirements

- Python 3.10 or newer

## How to Run

Heapsort demo:
```
python heapsort.py
```

Priority queue demo:
```
python priority_queue.py
```

Benchmarks:
```
python benchmark.py
```

## Notes

- Quicksort uses a simple last-element pivot; you may observe slower performance on sorted data.
- After running benchmarks, copy results into the table in `report.md`.
